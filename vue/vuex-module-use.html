# 应用命名空间使getters、mutation、action也相互独立。

例如form.js

如果希望你的模块具有更高的封装度和复用性，你可以通过添加 namespaced: true 的方式使其成为带命名空间的模块。当模块被注册后，它的所有 getter、action 及 mutation 都会自动根据模块注册的路径调整命名。

在组件中访问该module的getters -> this.$store.getters[‘form/*’]

在组件中访问该module的mutation-> this.$store.commit[‘form/*’]

在组件中访问该module的action->this.$store.dispatch[‘form/*’]
